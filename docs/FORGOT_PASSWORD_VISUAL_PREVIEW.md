## Component States

### 1. Initial State
- Empty email field
- Send button enabled
- Back to Login link

### 2. Typing State
- Email validation on blur
- Error message if invalid format
- Button stays enabled

### 3. Loading State
- Spinner in button
- "Sending email..." text
- Input disabled
- Button disabled

### 4. Success State
- Green success box
- Blue info box
- Back to Login button (primary)
- Resend email link

### 5. Error State
- Red error box
- Email field highlighted (if validation error)
- Send button enabled (retry)
